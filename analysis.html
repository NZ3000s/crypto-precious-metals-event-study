Binance Gold & Silver Perpetuals and ETF Markets

Author

KSE Python course project

1. Introduction and research question

Research question.
Did the introduction of Binance TRADFI gold and silver perpetual futures change volatility, liquidity and price integration in the corresponding precious metal ETF markets (GLD and SLV)?

Main hypotheses.

  • H1 (volatility): Realized volatility of gold and silver ETFs increased after the launch of Binance XAUUSDT and XAGUSDT TRADFI perpetual futures.
  • H2 (liquidity): Trading activity and liquidity of GLD and SLV improved (higher volume, lower Amihud illiquidity) after the launch.
  • H3 (integration): Short‑run price co‑movement and beta of GLD/SLV with Binance XAUUSDT/XAGUSDT returns increased after the launch.

For a more formal comparison we can run simple t‑tests.

from scipy import stats

def ttest_pre_post(series: pd.Series) -> dict:
    pre = series[df_core["post_event"] == 0].dropna()
    post = series[df_core["post_event"] == 1].dropna()
    if len(pre) < 3 or len(post) < 3:
        return {"t_stat": np.nan, "p_value": np.nan}
    t_stat, p_val = stats.ttest_ind(post, pre, equal_var=False)
    return {"t_stat": t_stat, "p_value": p_val}

def run_ols(y_col: str, x_cols: list, data: pd.DataFrame):
    """
    Run a simple OLS regression if there are enough observations.
    Returns a fitted model or None.
    """
    sub = data[[y_col] + x_cols].dropna()
    if sub.shape[0] < 10:
        print(f"Not enough non-missing observations for {y_col} (n={sub.shape[0]}). Skipping regression.")
        return None
    y = sub[y_col]
    X = sm.add_constant(sub[x_cols])
    model = sm.OLS(y, X, missing="drop")
    return model.fit()
